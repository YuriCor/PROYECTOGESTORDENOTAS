Titulo del proyecto
GESTOR DE NOTAS ACADEMICAS

Redacción del problema:
Es un sistema creado para que los estudiantes puedan gestionar fácilmente sus calificaciones en los diferentes cursos. Este sistema opera utilizando Python y esta dirigido a estudiantes que desean tener un control organizado y eficiente de sus notas, sin necesidad de usar aplicaciones mas complejas o bases de datos. Este permite registrar, buscar, modificar. eliminar y ordenar las calificaciones, facilitando el análisis y de los resultados académicos.

El objetivo principal es cubrir la necesidad de contar con un programa que permita gestionar múltiples opciones relacionadas con las notas, usando estructuras del lenguaje Python. Además incorpora funcionalidades como la búsqueda, el orden por notas y nombres, como el manejo de pilas y colas para registrar cambios y solicitudes. 

Requisitos del sistema 
FUNCIONALES
Registrar un nuevo curso y su nota correspondiente.
Mostrar todos los cursos y notas registrados.
Buscar un curso por nombre
Ordenar los cursos por nota y por nombre
Actualizar y eliminar notas de cursos existentes
Simular solicitudes de revisión mediante una cola y visualizar cambios recientes con una pila.

NO FUNCIONALES
El sistema se ejecutara en consola usando Python
No se usaran librerías externas, solo funcionalidades básicas
Esta programado con bucles y condicionales en pseudocodigo
